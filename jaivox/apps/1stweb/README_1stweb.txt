
This program works like the program in 1stsphinx, except that this one
uses Google's unofficial API for speech recognition. There are two
main programs here.

roadWeb.java: uses Google's unofficial API for speech synthesis.

roadWebFreetts.java: uses the Freetts package for synthesis.

To use the programs here:

roadWeb: compile roadWeb.java and all other java programs except
	roadWebFreetts. To run: java roadWeb.

roadWebFreetts: compile using sh compile.sh, run using sh run.sh.
